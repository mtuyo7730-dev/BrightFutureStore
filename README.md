# BrightFutureStore

Tienda de afiliados de Amazon, rápida y lista para GitHub Pages.

## Estructura
- `index.html` — página principal con categorías y productos.
- `style.css` — diseño oscuro/ligero, responsivo.
- `README.md` — notas de despliegue.

## Despliegue (GitHub Pages)
1. Sube estos tres archivos al repositorio público `BrightFutureStore`.
2. Configuración → Páginas → Construcción y despliegue:
   - Rama: `main`
   - Carpeta: `/ (root)`
